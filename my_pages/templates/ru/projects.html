API Weather Project

Учебный проект. Основной стек технологий:
Django, Vue Js, PostgreSql, API Openweather

Подготовлена заготовка для дальнейшей интеграции готового API погоды в будущие проекты. Сервер на Django принимает запрос на погоду в определенном регионе, городе. Если в базе нет данных проходит внутренний (на стороне сервера) запрос к API Openweather и данные сохранются в БД, потом отправляются как API ответ (JSON) клиенту. Для тестов развернут VUE JS шаблон с интерактивным вводом города пользователем.
